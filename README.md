
# Codeby - Front-end Challenge

***

## Rodando localmente

Para iniciar o projeto localmente, siga os seguintes passos:

### Frontend

- Instale as dependências:

```bash
yarn
ou
npm install
```
- De start no projeto:

```bash
yarn start
ou
npm start
```
Após esse processo, o frontend estará rodando na segunte url:
- <https://localhost:3000>
